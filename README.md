# Book-engine-search

## Description

This project is a book website that allows users to create and account and save books they like or want to read. The project uses apollo, mongo, and other such npm packages to provide a user experience.

## Installation

To install this application run npm i and install the neccesary npm packages.

## Usage 

To test this program yourself, clone the repository and once the packages are installed run the build.
